Історія криптовалют

Від концепції до глобального феномену - ключові моменти розвитку цифрових валют

2008

Народження Біткойна

Сатоші Накамото (псевдонім) опублікував white paper "Bitcoin: A Peer-to-Peer Electronic Cash System", в якомь описав концепцію децентралізованої цифрової валюти. Ця дата вважається початком ери криптовалют.

2009

Перший блок та перша транзакція

Згенеровано перший блок Біткойн (генезис-блок). Через кілька днів відбулася перша транзакція між Сатоші Накамото та програмістом Хелом Фінні.

2010

Перша реальна покупка

Програміст Ласло Ханєць купив дві піци за 10 000 BTC, що стало першою реальною покупкою за біткойни. Ця подія щороку відзначається як "День біткойн-піци".

2015

Запуск Ethereum

Віталік Бутерін та його команда запустили Ethereum - платформу для створення децентралізованих додатків на основі смарт-контрактів. Це відкрило нові можливості для блокчейн-технологій.

2021

Мейнстрімне визнання

Великі компанії, такі як Tesla та MicroStrategy, почали інвестувати в біткойн. Країни почали розробляти регуляторні рамки для криптовалют, а NFT набули масової популярності.

Ключова інформація

Основні поняття та технології, які варто знати про криптовалюти

Блокчейн

Децентралізована цифрова книга, яка записує всі транзакції в мережі. Кожен блок містить інформацію про попередній блок, створюючи ланцюг.

Смарт-контракти

Автоматизовані контракти, умови яких записані в коді. Виконуються автоматично при виконанні певних умов без посередників.

Майнінг

Процес створення нових блоків у блокчейні та підтвердження транзакцій. Майнери використовують обчислювальну потужність для розв'язання складних математичних задач.

Гаманці

Програмні або апаратні засоби для зберігання криптовалют. Містять приватні ключі, які дозволяють доступ до ваших цифрових активів.

Децентралізовані біржі (DEX)

Платформи для торгівлі криптовалютами без посередників. Користувачі торгують напряму один з одним через смарт-контракти.

NFT

Незмінні токени, які представляють унікальні цифрові або фізичні активи. Використовуються для мистецтва, колекційних предметів, ігрових активів тощо.
